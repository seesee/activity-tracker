はじめに

Activity Trackerは、スマート通知、包括的なレポート、Pomodoroタイマー機能を備えたパーソナルアクティビティトラッキングアプリケーションです。

主な機能：

  • 設定可能な作業/休憩サイクルを持つPomodoroタイマー
  • カスタマイズ可能な音声を持つスマート通知システム
  • カスタムテンプレートを持つ包括的なアクティビティレポート
  • ダークモードサポート
  • サービスワーカーによるオフライン機能
  • プログレッシブウェブアプリ（PWA）サポート

キーボードショートカット

  • Shift + スペース - アクティビティ入力フィールドにフォーカス
  • Shift + Enter - アクティビティフォームを送信
  • Ctrl + T - ToDo/非ToDoステータスを切り替え
  • Ctrl + N - ノートステータスを切り替え
  • Esc - モーダルを閉じる

アクティビティの追加

  • アクティビティフィールドに今していることを入力
  • 簡単な分類のために#ハッシュタグを追加
  • 完了が必要なタスクには「ToDoとしてマーク」ボタンを使用
  • 参考アイテムには「ノートとしてマーク」ボタンを使用（レポートから除外）
  • クイック設定ボタンを使用してToDoの期限を設定

検索とナビゲーション

  • グローバル検索を使用してアクティビティ、ToDo、ハッシュタグを検索
  • 🏷️ボタンをクリックして、ビジュアルクラウドでハッシュタグを閲覧
  • 任意のハッシュタグをクリックして関連アイテムを検索
  • ページネーション機能を使用して大きなリストをナビゲート

ToDoの管理

  • 専用の「To Do」セクションですべてのToDoを表示
  • ステータス、期限、カテゴリでフィルタリング
  • 「完了とマーク」をクリックしてToDoを終了
  • 期限切れアイテムは赤色で強調表示

レポート

  • 週次、カスタム日付範囲、またはテンプレートベースのレポートを生成
  • HTML、Markdown、またはCSV形式でレポートをエクスポート
  • 変数を使用したカスタムレポートテンプレートを作成
  • レポートに完了したToDoを含める/除外する

Pomodoroモード（オプション）

  • カスタマイズ可能な作業/休憩間隔でPomodoroテクニックを有効化
  • Pomodoroセッション中の自動アクティビティログ
  • 作業/休憩移行のためのオーディオ通知
  • 生産性維持を支援するフォーカスタイマー

設定とカスタマイズ

  • 通知間隔とアクティビティ時間をカスタマイズ
  • 複数の通知音から選択
  • ライト/ダークテーマを切り替え
  • より良いナビゲーションのためにページネーションサイズを設定
  • すべての設定は変更時に自動保存

データ管理

  • データをJSONバックアップとしてエクスポート
  • 以前にエクスポートしたデータをインポート
  • データはブラウザにローカルに保存
  • 初回ロード後はオフラインで動作
