시작하기

Activity Tracker는 스마트 알림, 포괄적인 보고서, Pomodoro 타이머 기능을 갖춘 개인 활동 추적 애플리케이션입니다.

주요 기능:

  • 구성 가능한 작업/휴식 주기를 가진 Pomodoro 타이머
  • 사용자 정의 가능한 사운드를 가진 스마트 알림 시스템
  • 사용자 정의 템플릿을 가진 포괄적인 활동 보고서
  • 다크 모드 지원
  • 서비스 워커를 이용한 오프라인 기능
  • Progressive Web App (PWA) 지원

키보드 단축키

  • Shift + 스페이스 - 활동 입력 필드에 포커스
  • Shift + Enter - 활동 양식 제출
  • Ctrl + T - 할 일/할 일 아님 상태 토글
  • Ctrl + N - 노트 상태 토글
  • Esc - 모달 닫기

활동 추가하기

  • 활동 필드에 하고 있는 일을 입력하세요
  • 쉬운 분류를 위해 #해시태그를 추가하세요
  • 완료가 필요한 작업에는 "할 일로 표시" 버튼을 사용하세요
  • 참조 항목에는 "노트로 표시" 버튼을 사용하세요 (보고서에서 제외됨)
  • 빠른 설정 버튼을 사용하여 할 일의 마감일을 설정하세요

검색 및 탐색

  • 전역 검색을 사용하여 활동, 할 일, 해시태그를 찾으세요
  • 🏷️ 버튼을 클릭하여 시각적 클라우드에서 해시태그를 탐색하세요
  • 관련 항목을 검색하려면 아무 해시태그나 클릭하세요
  • 큰 목록을 탐색하려면 페이지네이션 컨트롤을 사용하세요

할 일 관리

  • 전용 "할 일" 섹션에서 모든 할 일을 확인하세요
  • 상태, 마감일 또는 카테고리별로 필터링하세요
  • 할 일을 완료하려면 "완료로 표시"를 클릭하세요
  • 기한이 지난 항목은 빨간색으로 강조표시됩니다

보고서

  • 주간, 사용자 정의 날짜 범위 또는 템플릿 기반 보고서를 생성하세요
  • HTML, Markdown 또는 CSV 형식으로 보고서를 내보내세요
  • 변수가 있는 사용자 정의 보고서 템플릿을 생성하세요
  • 보고서에서 완료된 할 일을 포함/제외하세요

Pomodoro 모드 (선택사항)

  • 사용자 정의 가능한 작업/휴식 간격으로 Pomodoro 기법을 활성화하세요
  • Pomodoro 세션 중 자동 활동 로깅
  • 작업/휴식 전환을 위한 오디오 알림
  • 생산성을 유지하는 데 도움이 되는 집중 타이머

설정 및 사용자 정의

  • 알림 간격 및 활동 시간을 사용자 정의하세요
  • 여러 알림 사운드 중에서 선택하세요
  • 밝은/어두운 테마 간에 전환하세요
  • 더 나은 탐색을 위해 페이지네이션 크기를 설정하세요
  • 모든 설정은 변경할 때 자동으로 저장됩니다

데이터 관리

  • 데이터를 JSON 백업으로 내보내세요
  • 이전에 내보낸 데이터를 가져오세요
  • 데이터는 브라우저에 로컬로 저장됩니다
  • 초기 로드 후 오프라인에서 작동합니다
