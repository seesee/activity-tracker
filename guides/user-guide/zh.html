入门指南

活动追踪器是一个具有智能通知、全面报告和Pomodoro计时器功能的个人活动跟踪应用程序。

主要功能：

  • 可配置工作/休息周期的Pomodoro计时器
  • 带有可自定义声音的智能通知系统
  • 具有自定义模板的全面活动报告
  • 深色模式支持
  • 具有service worker的离线功能
  • 渐进式Web应用程序（PWA）支持

键盘快捷键

  • Shift + 空格 - 聚焦活动输入字段
  • Shift + Enter - 提交活动表单
  • Ctrl + T - 切换待办/非待办状态
  • Ctrl + N - 切换笔记状态
  • Esc - 关闭模态框

添加活动

  • 在活动字段中输入您正在做的事情
  • 添加#标签以便轻松分类
  • 使用"标记为待办"按钮标记需要完成的任务
  • 使用"标记为笔记"按钮标记参考项目（从报告中排除）
  • 使用快速设置按钮为待办事项设置截止日期

搜索和导航

  • 使用全局搜索查找活动、待办事项和标签
  • 点击🏷️按钮在视觉云中浏览标签
  • 点击任何标签搜索相关项目
  • 使用分页控件浏览大型列表

管理待办事项

  • 在专用的"待办"部分查看所有待办事项
  • 按状态、截止日期或类别筛选
  • 点击"标记完成"完成待办事项
  • 逾期项目以红色突出显示

报告

  • 生成每周、自定义日期范围或基于模板的报告
  • 导出HTML、Markdown或CSV格式的报告
  • 创建带有变量的自定义报告模板
  • 在报告中包含/排除已完成的待办事项

Pomodoro模式（可选）

  • 启用具有可自定义工作/休息间隔的Pomodoro技术
  • Pomodoro会话期间自动记录活动
  • 工作/休息转换的音频通知
  • 专注计时器帮助保持生产力

设置和自定义

  • 自定义通知间隔和活动时间
  • 从多种通知声音中选择
  • 在明/暗主题之间切换
  • 设置分页大小以便更好地导航
  • 所有设置在您更改时自动保存

数据管理

  • 将您的数据导出为JSON备份
  • 导入先前导出的数据
  • 数据存储在您的浏览器本地
  • 初次加载后可离线工作
