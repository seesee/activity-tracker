Ξεκινώντας

Το Activity Tracker είναι μια εφαρμογή παρακολούθησης προσωπικών δραστηριοτήτων με έξυπνες ειδοποιήσεις, ολοκληρωμένες αναφορές και λειτουργικότητα χρονομέτρου Pomodoro.

Κύρια χαρακτηριστικά:

  • Χρονόμετρο Pomodoro με διαμορφώσιμους κύκλους εργασίας/διαλείμματος
  • Έξυπνο σύστημα ειδοποιήσεων με προσαρμόσιμους ήχους
  • Ολοκληρωμένες αναφορές δραστηριοτήτων με προσαρμοσμένα πρότυπα
  • Υποστήριξη σκοτεινής λειτουργίας
  • Λειτουργικότητα offline με service worker
  • Υποστήριξη Progressive Web App (PWA)

Συντομεύσεις Πληκτρολογίου

  • Shift + Space - Εστίαση στο πεδίο εισαγωγής δραστηριότητας
  • Shift + Enter - Υποβολή φόρμας δραστηριότητας
  • Ctrl + T - Εναλλαγή κατάστασης todo/όχι todo
  • Ctrl + N - Εναλλαγή κατάστασης σημείωσης
  • Esc - Κλείσιμο modal

Προσθήκη Δραστηριοτήτων

  • Εισάγετε τι κάνετε στο πεδίο δραστηριότητας
  • Προσθέστε #hashtags για εύκολη κατηγοριοποίηση
  • Χρησιμοποιήστε το κουμπί "Σήμανση ως Todo" για εργασίες που χρειάζονται ολοκλήρωση
  • Χρησιμοποιήστε το κουμπί "Σήμανση ως Σημείωση" για στοιχεία αναφοράς (εξαιρούνται από αναφορές)
  • Ορίστε ημερομηνίες λήξης για todos χρησιμοποιώντας κουμπιά γρήγορης ρύθμισης

Αναζήτηση και Πλοήγηση

  • Χρησιμοποιήστε την καθολική αναζήτηση για να βρείτε δραστηριότητες, todos και hashtags
  • Κάντε κλικ στο κουμπί 🏷️ για να περιηγηθείτε στα hashtags σε ένα οπτικό σύννεφο
  • Κάντε κλικ σε οποιοδήποτε hashtag για να αναζητήσετε σχετικά στοιχεία
  • Χρησιμοποιήστε τα στοιχεία ελέγχου σελιδοποίησης για να πλοηγηθείτε σε μεγάλες λίστες

Διαχείριση Todos

  • Προβολή όλων των todos στην αφιερωμένη ενότητα "Προς Εκτέλεση"
  • Φιλτράρισμα κατά κατάσταση, ημερομηνία λήξης ή κατηγορία
  • Κάντε κλικ "Σήμανση Ολοκληρωμένου" για να τελειώσετε todos
  • Τα καθυστερημένα στοιχεία είναι επισημασμένα με κόκκινο

Αναφορές

  • Δημιουργία εβδομαδιαίων, προσαρμοσμένου εύρους ημερομηνιών ή βασισμένων σε πρότυπα αναφορών
  • Εξαγωγή αναφορών σε μορφές HTML, Markdown ή CSV
  • Δημιουργία προσαρμοσμένων προτύπων αναφοράς με μεταβλητές
  • Συμπερίληψη/εξαίρεση ολοκληρωμένων todos από αναφορές

Λειτουργία Pomodoro (Προαιρετική)

  • Ενεργοποίηση τεχνικής Pomodoro με προσαρμόσιμα διαστήματα εργασίας/διαλείμματος
  • Αυτόματη καταγραφή δραστηριοτήτων κατά τη διάρκεια των συνεδριών Pomodoro
  • Ηχητικές ειδοποιήσεις για μεταβάσεις εργασίας/διαλείμματος
  • Χρονόμετρο εστίασης για τη διατήρηση της παραγωγικότητας

Ρυθμίσεις και Προσαρμογή

  • Προσαρμογή διαστημάτων ειδοποιήσεων και ωρών δραστηριότητας
  • Επιλογή από πολλούς ήχους ειδοποιήσεων
  • Εναλλαγή μεταξύ φωτεινών/σκοτεινών θεμάτων
  • Ορισμός μεγέθους σελιδοποίησης για καλύτερη πλοήγηση
  • Όλες οι ρυθμίσεις αυτόματα-αποθηκεύονται καθώς τις αλλάζετε

Διαχείριση Δεδομένων

  • Εξαγωγή των δεδομένων σας ως αντίγραφο ασφαλείας JSON
  • Εισαγωγή προηγουμένως εξαχθέντων δεδομένων
  • Τα δεδομένα αποθηκεύονται τοπικά στον περιηγητή σας
  • Λειτουργεί offline μετά την αρχική φόρτωση
