Cychwyn

Mae templedi'n defnyddio cystrawen amnewid newidyn syml gyda bracedi cyrliog dwbl i greu adroddiadau deinamig.

Cystrawen sylfaenol

  • {{variable}} - Amnewid newidyn syml
  • {{#each array}} ... {{/each}} - Dolennu drwy araeau
  • {{#if condition}} ... {{/if}} - Cynnwys amodol

Cystrawen bibell a chadwyno swyddogaethau

Trawsnewidiwch ddata gan ddefnyddio cystrawen pibell (|) i gadwyno swyddogaethau lluosog gyda'i gilydd:

Cystrawen bibell sylfaenol
  • {{ variable | function }} - Rhoi un trawsnewidiad ar waith
  • {{ variable | function1 | function2 }} - Cadwyno trawsnewidiadau lluosog
  • {{ variable | function('parameter') }} - Trosglwyddo paramedrau i swyddogaethau
Swyddogaethau trawsnewid sydd ar gael
  • date('format') - Fformatio stampiau amser fel dyddiadau
    • {{ timestamp | date('yyyy-mm-dd') }} → "2025-08-02"
    • {{ timestamp | date('mm/dd/yyyy') }} → "08/02/2025"
    • {{ timestamp | date('dd mmm yyyy') }} → "02 Aug 2025"
  • time - Echdynnu amser o stampiau amser
    • {{ timestamp | time }} → "14:30:25"
  • escapeCsv - Dianc cymeriadau arbennig CSV
    • Lapio testun mewn dyfynodau os yw'n cynnwys atalnodau, dyfynodau neu doriadau llinell
    • {{ "Hello, World" | escapeCsv }} → "\"Hello, World\""
  • stripLinebreaks - Tynnu toriadau llinell o destun
    • {{ "Line 1\nLine 2" | stripLinebreaks }} → "Line 1 Line 2"
Enghraifft templed CSV gyda chadwyno

Dyma sut mae'r templed CSV yn defnyddio cadwyno i greu allbwn glân wedi'i ddianc yn gywir:

{{#each entry = entries}}{{ entry.timestamp | date('yyyy-mm-dd') }},{{ entry.timestamp | time }},{{ entry.activity | escapeCsv | stripLinebreaks }},{{ entry.description | escapeCsv | stripLinebreaks }}
{{/each}}

Sut mae hyn yn gweithio:

  • entry.timestamp | date('yyyy-mm-dd') - Trosi'r stamp amser i fformat YYYY-MM-DD
  • entry.timestamp | time - Echdynnu rhan yr amser
  • entry.activity | escapeCsv | stripLinebreaks - Tynnu toriadau llinell gyntaf, yna dianc ar gyfer CSV
  • entry.description | escapeCsv | stripLinebreaks - Yr un broses ar gyfer disgrifiadau
Mae trefn yn bwysig mewn cadwyno!

Mae swyddogaethau'n cael eu cymhwyso o'r chwith i'r dde, felly mae trefn yn bwysig:

  • ✅ {{ text | stripLinebreaks | escapeCsv }} - Tynnu toriadau gyntaf, yna dianc
  • ❌ {{ text | escapeCsv | stripLinebreaks }} - Dianc gyntaf gall ymyrryd â thynnu toriadau

Data sydd ar gael

Mae'r data canlynol ar gael yn eich templedi:

  • {{reportTitle}} - Teitl yr adroddiad
  • {{startDate}} - Dyddiad dechrau'r adroddiad (wedi'i fformatio)
  • {{endDate}} - Dyddiad diwedd yr adroddiad (wedi'i fformatio)
  • {{totalEntries}} - Cyfanswm nifer cofnodion gweithgaredd
  • {{entries}} - Arae o holl gofnodion gweithgaredd

Priodweddau cofnod

Mae gan bob cofnod yn y ddolen {{#each entries}}:

  • {{activity}} - Enw'r gweithgaredd
  • {{description}} - Disgrifiad y gweithgaredd
  • {{timestamp}} - Stamp amser ISO crai
  • {{formattedTime}} - Amser wedi'i fformatio'n daclus (e.e. "2:30 PM")
  • {{formattedDate}} - Dyddiad wedi'i fformatio'n daclus (e.e. "Jan 15, 2025")
  • {{isTodo}} - Booleaidd: true os yw hwn yn eitem todo
  • {{tags}} - Arae o hashnodau ar gyfer y cofnod hwn

Trawsnewidiadau cyffredin

Trawsnewid stampiau amser a fformatio data:

  • Fformatau dyddiad: Defnyddiwch {{formattedDate}} a {{formattedTime}} ar gyfer fformatau darllenadwy
  • Cynnwys amodol: {{#if description}}{{description}}{{/if}} - dangos dim ond os oes disgrifiad
  • Dolennu drwy dagiau: {{#each tags}}#{{this}} {{/each}} - dangos pob hashnod
  • Hidlo todo: {{#if isTodo}}TODO: {{/if}} - marcio eitemau todo

Fformatio Markdown

Defnyddiwch gystrawen Markdown ar gyfer fformatio darllenadwy:

  • # {{reportTitle}} - Pennawd mawr
  • ## {{activity}} - Pennawd canolig
  • **{{activity}}** - Testun trwm
  • *{{description}}* - Testun italig
  • - {{activity}} - Pwynt bwled
  • `{{formattedTime}}` - Fformatio cod

Enghraifft templed

# {{reportTitle}}
O {{startDate}} i {{endDate}}

Cyfanswm gweithgareddau: {{totalEntries}}

{{#each entries}}
## {{formattedTime}} - {{activity}}
{{#if description}}
*{{description}}*
{{/if}}
{{#if tags}}
Tagiau: {{#each tags}}#{{this}} {{/each}}
{{/if}}
{{#if isTodo}}
**EITEM TODO**
{{/if}}

{{/each}}

Awgrymiadau proffesiynol

  • Templedi HTML: Defnyddiwch ar gyfer fformatio cyfoethog gydag arddulliau CSS a chynlluniau cymhleth
  • Templedi Markdown: Perffaith ar gyfer dogfennaeth syml a darllenadwy
  • Templedi CSV: Delfrydol ar gyfer dadansoddi data mewn taenlenni
  • Nodwedd rhagolwg: Profwch eich templedi bob tro cyn cadw
  • Fformatau cyfaddas: Cyfunwch newidynnau lluosog ar gyfer cynlluniau unigryw
