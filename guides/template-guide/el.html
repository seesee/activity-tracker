Ξεκινώντας

Τα πρότυπα χρησιμοποιούν μια απλή σύνταξη αντικατάστασης μεταβλητών με διπλές άγκιστρα για τη δημιουργία δυναμικών αναφορών.

Βασική σύνταξη

  • {{variable}} - Απλή αντικατάσταση μεταβλητής
  • {{#each array}} ... {{/each}} - Βρόχος μέσω πινάκων
  • {{#if condition}} ... {{/if}} - Υπό συνθήκη περιεχόμενο

Σύνταξη pipe και αλυσίδωση συναρτήσεων

Μεταμορφώστε δεδομένα χρησιμοποιώντας τη σύνταξη pipe (|) για να αλυσιδώσετε πολλές συναρτήσεις μαζί:

Βασική σύνταξη pipe
  • {{ variable | function }} - Εφαρμόστε ένα μετασχηματισμό
  • {{ variable | function1 | function2 }} - Αλυσιδώστε πολλούς μετασχηματισμούς
  • {{ variable | function('parameter') }} - Περάστε παραμέτρους σε συναρτήσεις
Διαθέσιμες συναρτήσεις μετασχηματισμού
  • date('format') - Μορφοποίηση χρονικών σημάνσεων ως ημερομηνίες
    • {{ timestamp | date('yyyy-mm-dd') }} → "2025-08-02"
    • {{ timestamp | date('mm/dd/yyyy') }} → "08/02/2025"
    • {{ timestamp | date('dd mmm yyyy') }} → "02 Aug 2025"
  • time - Εξαγωγή ώρας από χρονικές σημάνσεις
    • {{ timestamp | time }} → "14:30:25"
  • escapeCsv - Διαφυγή ειδικών χαρακτήρων CSV
    • Τυλίγει το κείμενο σε εισαγωγικά αν περιέχει κόμματα, εισαγωγικά ή αλλαγές γραμμής
    • {{ "Hello, World" | escapeCsv }} → "\"Hello, World\""
  • stripLinebreaks - Αφαίρεση αλλαγών γραμμής από κείμενο
    • {{ "Line 1\nLine 2" | stripLinebreaks }} → "Line 1 Line 2"
Παράδειγμα προτύπου CSV με αλυσίδωση

Πώς το πρότυπο CSV χρησιμοποιεί αλυσίδωση για να δημιουργήσει καθαρό, σωστά escaped έξοδο:

{{#each entry = entries}}{{ entry.timestamp | date('yyyy-mm-dd') }},{{ entry.timestamp | time }},{{ entry.activity | escapeCsv | stripLinebreaks }},{{ entry.description | escapeCsv | stripLinebreaks }}
{{/each}}

Πώς λειτουργεί:

  • entry.timestamp | date('yyyy-mm-dd') - Μετατρέπει τη χρονική σήμανση σε μορφή YYYY-MM-DD
  • entry.timestamp | time - Εξάγει το τμήμα της ώρας
  • entry.activity | escapeCsv | stripLinebreaks - Πρώτα αφαιρεί αλλαγές γραμμής, στη συνέχεια escape για CSV
  • entry.description | escapeCsv | stripLinebreaks - Ίδια διαδικασία για περιγραφές
Η σειρά είναι σημαντική στην αλυσίδωση!

Οι συναρτήσεις εφαρμόζονται από αριστερά προς δεξιά, οπότε η σειρά είναι σημαντική:

  • ✅ {{ text | stripLinebreaks | escapeCsv }} - Αφαιρέστε αλλαγές πρώτα, στη συνέχεια escape
  • ❌ {{ text | escapeCsv | stripLinebreaks }} - Το escape πρώτα μπορεί να παρεμβάλει στην αφαίρεση αλλαγών

Διαθέσιμα δεδομένα

Τα ακόλουθα δεδομένα είναι διαθέσιμα στα πρότυπά σας:

  • {{reportTitle}} - Ο τίτλος της αναφοράς
  • {{startDate}} - Ημερομηνία έναρξης αναφοράς (μορφοποιημένη)
  • {{endDate}} - Ημερομηνία λήξης αναφοράς (μορφοποιημένη)
  • {{totalEntries}} - Συνολικός αριθμός καταχωρήσεων δραστηριότητας
  • {{entries}} - Πίνακας όλων των καταχωρήσεων δραστηριότητας

Ιδιότητες καταχωρήσεων

Κάθε καταχώρηση στον βρόχο {{#each entries}} έχει:

  • {{activity}} - Το όνομα της δραστηριότητας
  • {{description}} - Περιγραφή δραστηριότητας
  • {{timestamp}} - Ακατέργαστη χρονική σήμανση ISO
  • {{formattedTime}} - Όμορφα μορφοποιημένη ώρα (π.χ. "2:30 PM")
  • {{formattedDate}} - Όμορφα μορφοποιημένη ημερομηνία (π.χ. "Jan 15, 2025")
  • {{isTodo}} - Boolean: true αν αυτό είναι στοιχείο todo
  • {{tags}} - Πίνακας hashtags για αυτή την καταχώρηση

Συνήθεις μετασχηματισμοί

Μεταμόρφωση χρονικών σημάνσεων και μορφοποίηση δεδομένων:

  • Μορφές ημερομηνίας: Χρησιμοποιήστε {{formattedDate}} και {{formattedTime}} για αναγνώσιμες μορφές
  • Υπό συνθήκη περιεχόμενο: {{#if description}}{{description}}{{/if}} - εμφάνιση μόνο αν υπάρχει περιγραφή
  • Βρόχος μέσω tags: {{#each tags}}#{{this}} {{/each}} - εμφάνιση όλων των hashtags
  • Φιλτράρισμα todo: {{#if isTodo}}TODO: {{/if}} - σήμανση στοιχείων todo

Μορφοποίηση Markdown

Χρησιμοποιήστε σύνταξη Markdown για αναγνώσιμη μορφοποίηση:

  • # {{reportTitle}} - Μεγάλος τίτλος
  • ## {{activity}} - Μεσαίος τίτλος
  • **{{activity}}** - Έντονο κείμενο
  • *{{description}}* - Πλάγιο κείμενο
  • - {{activity}} - Κουκκίδα
  • `{{formattedTime}}` - Μορφοποίηση κώδικα

Παράδειγμα προτύπου

# {{reportTitle}}
Από {{startDate}} έως {{endDate}}

Συνολικές δραστηριότητες: {{totalEntries}}

{{#each entries}}
## {{formattedTime}} - {{activity}}
{{#if description}}
*{{description}}*
{{/if}}
{{#if tags}}
Tags: {{#each tags}}#{{this}} {{/each}}
{{/if}}
{{#if isTodo}}
**ΣΤΟΙΧΕΙΟ TODO**
{{/if}}

{{/each}}

Συμβουλές επαγγελματία

  • Πρότυπα HTML: Χρησιμοποιήστε για πλούσια μορφοποίηση με στυλ CSS και πολύπλοκες διατάξεις
  • Πρότυπα Markdown: Τέλεια για απλή, αναγνώσιμη τεκμηρίωση
  • Πρότυπα CSV: Ιδανικά για ανάλυση δεδομένων σε υπολογιστικά φύλλα
  • Λειτουργία προεπισκόπησης: Πάντα δοκιμάζετε τα πρότυπά σας πριν αποθηκεύσετε
  • Προσαρμοσμένες μορφές: Συνδυάστε πολλές μεταβλητές για μοναδικές διατάξεις
